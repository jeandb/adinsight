# Story 02.3: Remover tabela de pedidos individuais da página de Faturamento

**Status:** Done
**Epic:** Revenue Dashboard
**Traces to:** PRD Seção 8 — Integração WooCommerce / Dashboard de Faturamento

## User Story

Como gestora de tráfego ou diretora, quero ver apenas métricas consolidadas na página de Faturamento, sem a lista de pedidos individuais, para que o dashboard seja focado em análise de performance e não em operação.

## Acceptance Criteria

- [x] A tabela de pedidos individuais (`OrdersTable`) é removida da página `/revenue`
- [x] Os cards de resumo por status (Story 02.2) permanecem visíveis
- [x] Nenhum outro componente da página é afetado
- [x] A query `wooStoresApi.listOrders` deixa de ser chamada no carregamento da página

## Technical Notes

- Remover o componente `<OrdersTable>` e sua chamada em `RevenuePage.tsx`
- Remover o import de `wooStoresApi` se não for mais utilizado em outro lugar na página
- O componente `OrdersTable` em si pode ser mantido no arquivo (não apagar) caso seja reaproveitado em outra rota futura
